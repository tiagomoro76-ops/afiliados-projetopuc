# Mini Shopee Afiliado

Projeto Python para listar produtos de um nicho e calcular comissão de afiliados.

## Funcionalidades
- Lista produtos com preço e categoria
- Calcula comissão automática de 10%
- Permite filtrar produtos por categoria

## Como usar
1. Execute `app.py`
2. Digite uma categoria para filtrar ou pressione ENTER para ver todos os produtos
